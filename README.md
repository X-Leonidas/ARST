## ARST介绍
+ 陈皓推行的ARST活动，即每周至少做一个 leetcode 的算法题、阅读并点评至少一篇英文技术文章、学习至少一个技术技巧、分享一篇有观点和思考的技术文章。
（也就是 Algorithm、Review、Tip、Share 简称 ARTS）你需要坚持至少一年。
